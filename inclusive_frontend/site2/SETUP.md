# Инструкция по установке и запуску

## Требования

- Node.js 18+ 
- npm или yarn

## Шаги установки

1. **Установите зависимости:**
```bash
npm install
```

2. **Настройте переменные окружения:**
   
   Файл `.env.local` уже создан с вашими API ключами:
   - `NEXT_PUBLIC_OPENROUTER_API_KEY` - для AI помощника
   - `NEXT_PUBLIC_YANDEX_MAP_API_KEY` - для Яндекс карт

3. **Запустите dev сервер:**
```bash
npm run dev
```

4. **Откройте браузер:**
   Перейдите на [http://localhost:3000](http://localhost:3000)

## Сборка для production

```bash
npm run build
npm start
```

## Структура проекта

- `/app` - страницы Next.js (App Router)
- `/components` - React компоненты
- `/lib` - утилиты, API, store
- `/types` - TypeScript типы

## Основные функции

### Регистрация и вход
- Регистрация для сотрудников и работодателей
- Вход по телефону/почте и паролю

### Для сотрудников
- Поиск вакансий с фильтрами
- Генератор резюме (AI)
- Загрузка PDF резюме
- Карта с вакансиями
- Форум
- История заявок

### Для работодателей
- Размещение вакансий
- AI генератор инклюзивных описаний
- Просмотр резюме
- Принятие/отклонение заявок

### Премиум функции
- Basic: онлайн собеседование, улучшенный AI
- Plus: именной бейдж, сертификация

## Демо данные

При первом входе в систему автоматически загружаются 3 демо-вакансии для тестирования.

## Поддержка

При возникновении проблем проверьте:
1. Правильность API ключей в `.env.local`
2. Установлены ли все зависимости (`npm install`)
3. Запущен ли dev сервер (`npm run dev`)
